Guía de Mantenimiento

Esta guía explica cómo agregar y modificar contenido en la página web de Vamos Somos Chilenos. No necesitas saber programar — solo seguir las instrucciones y copiar/pegar los bloques de código. El único archivo que necesitas editar es index.html. Ábrelo con cualquier editor de texto (VS Code, Notepad++, o incluso el Bloc de Notas).

Archivos del Proyecto

Archivo Para qué sirve ¿Hay que editarlo?
index.html Toda la página (contenido, textos, links) Sí — aquí van todos los cambios
css/styles.css Los estilos visuales (colores, tamaños) Solo si quieres cambiar colores o diseño
js/main.js Animaciones y funcionalidad Normalmente no
assets/images/ Fotos del equipo y logos de sponsors Agregar imágenes aquí

Cambiar el Video Principal

El video principal es un embed de YouTube en la sección "En Vivo". Para cambiarlo:

  1. Abre el video en YouTube y copia el ID del video desde la URL. El ID es la parte después de v=. Por ejemplo, en youtube.com/watch?v=NS6wsDt7Qwo el ID es NS6wsDt7Qwo.
  2. En index.html, busca esta línea (alrededor de la línea 111):
<iframe
    src="https://www.youtube.com/embed/NS6wsDt7Qwo"
index.html ~ línea 111
  1. Reemplaza NS6wsDt7Qwo por el nuevo ID del video. Ejemplo:
<iframe
    src="https://www.youtube.com/embed/NUEVO_ID_AQUI"
resultado
Para buscar rápido en tu editor: presiona Ctrl + F (o Cmd + F en Mac) y busca youtube.com/embed.

Agregar un Nuevo Episodio al Carousel

Los episodios están en la sección "Episodios Recientes". Para agregar uno nuevo:

  1. Busca en index.html la sección del carousel (busca carousel__track, alrededor de la línea 215).
  2. Copia el siguiente bloque y pégalo al inicio, justo después de <div class="carousel__track"> para que aparezca primero:
<!-- NUEVO EPISODIO -->
<article class="carousel__card">
    <div class="carousel__thumb">
        <img src="https://img.youtube.com/vi/ID_DEL_VIDEO/mqdefault.jpg"
             alt="Descripción del episodio" loading="lazy">
        <span class="carousel__play"><i class="fa-solid fa-play"></i></span>
    </div>
    <div class="carousel__info">
        <h4>Título del Episodio</h4>
        <time>Viernes 19:00 hrs</time>
    </div>
</article>
copiar y pegar
  1. Cambia las 3 partes resaltadas:
    • ID_DEL_VIDEO → el ID de YouTube (la miniatura se genera automáticamente)
    • Título del Episodio → nombre descriptivo
    • Viernes 19:00 hrs → día y hora correspondiente
La miniatura del video se genera sola con la URL https://img.youtube.com/vi/ID/mqdefault.jpg. No necesitas descargar ninguna imagen.
Si el carousel tiene muchos episodios (más de 8), considera eliminar los más antiguos para que no quede muy largo. Borra el bloque <article class="carousel__card">...</article> completo del episodio que quieras quitar.

Agregar un Auspiciador

Opción A: Auspiciador con logo (imagen)

  1. Guarda el logo en la carpeta assets/images/sponsors/. Formatos recomendados: PNG con fondo transparente, tamaño ideal 200x100px.
  2. Busca sponsors__grid en index.html (alrededor de la línea 385).
  3. Agrega este bloque dentro del grid:
<div class="sponsors__logo">
    <img src="assets/images/sponsors/nombre-logo.png"
         alt="Nombre del Auspiciador"
         loading="lazy">
    <span>Nombre del Auspiciador</span>
</div>
copiar y pegar

Opción B: Auspiciador sin logo (solo icono + texto)

Si aún no tienes el logo, puedes usar un icono temporal:

<div class="sponsors__logo">
    <i class="fa-solid fa-building"></i>
    <span>Nombre Empresa</span>
</div>
copiar y pegar

Quitar un auspiciador placeholder

Para eliminar los auspiciadores de ejemplo, busca y borra el bloque completo <div class="sponsors__logo">...</div> que corresponda.

Los logos aparecen en escala de grises y toman color al pasar el mouse por encima. Este efecto es automático. Si usas imágenes, asegúrate de que sean en color (el CSS se encarga del efecto).

Editar Información del Equipo

Las tarjetas del equipo están en la sección "Equipo" (busca equipo__grid, alrededor de la línea 327).

Cambiar nombre, rol o bio

Busca la tarjeta de la persona y edita directamente:

<div class="equipo__card">
    <div class="equipo__avatar">
        <i class="fa-solid fa-user"></i>
    </div>
    <h4 class="equipo__name">Nombre</h4>
    <span class="equipo__role">Conductor</span>
    <p class="equipo__bio">Descripción corta de la persona.</p>
</div>
editar textos

Agregar un nuevo miembro

Copia una tarjeta completa <div class="equipo__card">...</div> y pégala dentro del grid correspondiente (viernes o sábado). Cambia nombre, rol y bio.

Agregar Fotos del Equipo

Actualmente cada miembro tiene un icono genérico. Para poner fotos reales:

  1. Guarda la foto en assets/images/hosts/. Recomendación: foto cuadrada, mínimo 200x200px, formato JPG o PNG.
  2. En la tarjeta de la persona, reemplaza el icono por una imagen:

Antes (icono):

<div class="equipo__avatar">
    <i class="fa-solid fa-user"></i>
</div>
antes

Después (foto):

<div class="equipo__avatar">
    <img src="assets/images/hosts/rodrigo.jpg"
         alt="Rodrigo"
         loading="lazy">
</div>
después
Para que la foto se vea circular, agrega esta línea en css/styles.css (búscalo al final del bloque .equipo__avatar):

.equipo__avatar img { width: 100%; height: 100%; object-fit: cover; border-radius: 50%; }

Cambiar Horarios

Los horarios aparecen en 3 lugares. Debes actualizarlos todos:

Ubicación Qué buscar en index.html Línea aprox.
Hero (parte principal) hero__schedule-time ~85
Sección En Vivo envivo__horario ~130
Sección Equipo (subtítulos) equipo__subtitle ~326
Recuerda actualizar los 3 lugares para que la información sea consistente en toda la página.

Actualizar Links de Redes Sociales

Los links de redes aparecen en 2 lugares:

  • Sección "Nuestras Redes" — tarjetas grandes (buscar redes__card, línea ~160)
  • Footer — iconos pequeños (buscar footer__social-links, línea ~496)

Para cambiar un link, busca la URL actual y reemplázala por la nueva. Ejemplo:

<!-- Cambiar el link de YouTube -->
<a href="https://youtube.com/@NuevoCanal"
   target="_blank" rel="noopener noreferrer">
ejemplo
Siempre mantén target="_blank" y rel="noopener noreferrer" en los links externos. Esto hace que se abran en una pestaña nueva de forma segura.

Configurar el Formulario de Contacto

El formulario usa Formspree, un servicio gratuito que recibe los mensajes y los envía a tu email.

  1. Ve a formspree.io y crea una cuenta gratis.
  2. Crea un nuevo formulario y copia el ID que te dan (algo como xabcdefg).
  3. En index.html, busca YOUR_FORM_ID (línea ~434) y reemplázalo:

Antes:

<form action="https://formspree.io/f/YOUR_FORM_ID"
antes

Después:

<form action="https://formspree.io/f/xabcdefg"
después
El plan gratuito de Formspree permite hasta 50 envíos por mes, más que suficiente para empezar.

Subir Cambios a GitHub Pages

Después de hacer cambios en los archivos, sube las actualizaciones a GitHub para que se vean en la web:

# 1. Abre la terminal en la carpeta del proyecto
cd VamosSomosChilenos

# 2. Agrega los archivos modificados
git add .

# 3. Crea un commit describiendo qué cambiaste
git commit -m "Agrego nuevo episodio y actualizo auspiciadores"

# 4. Sube los cambios
git push
terminal / consola

Los cambios aparecerán en la web en 1-2 minutos después del push.

Si no usas la terminal, puedes editar los archivos directamente en github.com: abre el archivo, haz clic en el lápiz (editar), haz tus cambios y presiona "Commit changes".

Errores Comunes

Problema Causa probable Solución
La página se ve rota Falta cerrar una etiqueta HTML (</div>, </article>) Revisa que cada etiqueta abierta tenga su cierre
El video no aparece ID de YouTube incorrecto Verifica que el ID sea correcto (11 caracteres)
La imagen no carga Ruta del archivo mal escrita Verifica que el nombre coincida exactamente (mayúsculas/minúsculas importan)
El formulario no envía Falta configurar Formspree Reemplaza YOUR_FORM_ID con tu ID real
Los cambios no se ven en la web No hiciste git push o el caché Haz push y espera 2 min. Prueba con Ctrl+Shift+R
Antes de subir cambios, siempre abre index.html directamente en tu navegador (doble clic en el archivo) para verificar que todo se vea bien.
